# 準備するもの

## X

キャラクターごとに確認します。

- Xアカウント
- X Developer App
- OAuth 2.0 Client ID
- OAuth 2.0 Client Secret
- App permissions: Read and write
- Callback URL: `https://公開URL/x-callback`
- Website URL: `https://公開URL/`

## WordPress

キャラクターごとに確認します。

- サイトURL
- 投稿用ユーザー名
- Application Password
- 投稿カテゴリー
- アイキャッチURL
- シリーズ名
- スラッグ開始番号

## OpenAI

- OpenAI API Key
- 画像生成モデルの利用可否

## ホスティング

最初はRenderが移しやすいです。

- Node.js Web Service
- Persistent Disk
- Environment Variables
- 公開URL

## Obsidian

完全オンライン化では、スマホから直接ローカルObsidianへ保存することはできません。
初期版ではオンラインサーバー上にMarkdownを保存し、あとでPC側へ同期する構成にします。

候補:

- GitHubへMarkdownを保存
- Google Drive / Dropbox / OneDrive同期
- Supabase Storageに保存してPCで同期
